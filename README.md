2. COPY folder (VIEWS) inside APPLICATION/VIEWS to the directory ex (C:/XAMPP/htdocs/application/)
3. COPY folder (Modal) inside APPLICATION/MODALS to the directory ex (C:/XAMPP/htdocs/application/)
